# 📌 Project Overview

The Employee Management System is a CRUD-based web application that allows users to:

- Add new employees
- View employee details
- Update employee information
- Delete employee records
- Manage employee data through REST APIs
- Experience a responsive and professional dashboard UI

The project follows a full-stack architecture with a React frontend and Spring Boot backend connected to a MySQL database.
